The script for this plugin are in [Invocation/Editor]

There is an Example scene that has a simple animation script to 
help demo the capablitiy of this plugin. Under Window you will find 
a menu item named Invoker this will open a Inpector like panel. Using 
this panel all functions of the monobehaviour components attached to 
the selected gameobject will be display and can be invoked. Basic 
parameter types can be assigned, other complex types will be set 
to either null or default.

This System supports C#, support for other languages can not be
guaranteed.
